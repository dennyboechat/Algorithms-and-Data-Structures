# Algorithms and Data Structures Assignment 6

This repository contains implementations and experiments related to algorithms and data structures for MSCS532 Assignment 6. The project is organized into the following main folders:

## Data Structures Implementation
- **data_structures.py**: Contains Python implementations of fundamental data structures used in the assignment.
- **README.md**: Documentation for the data structures module.

## Selection Algorithms
- **algorithm_comparison.py**: Script to compare the performance of different selection algorithms.
- **examples.py**: Example usages and demonstrations of the selection algorithms.
- **median_of_medians.py**: Implementation of the Median of Medians selection algorithm.
- **randomized_quickselect.py**: Implementation of the Randomized Quickselect algorithm.
- **simple_demo.py**: Simple demonstration script for selection algorithms.
- **test_median_of_medians.py**: Unit tests for the Median of Medians algorithm.
- **test_randomized_quickselect.py**: Unit tests for the Randomized Quickselect algorithm.
- **README.md**: Documentation for the selection algorithms module.

## Report
- **Report.docx**: Contains a detailed report of the assignment, including methodology, results, and conclusions.

For more information, please refer to the README files in each folder and the detailed report in `Report.docx`.
